## Langkah 1: Upload ke GitHub

1. Buat repository baru di GitHub
2. Upload 3 file ini ke repository:
   - `index.html`
   - `script.js`
   - `README.md`

## Langkah 2: Aktifkan GitHub Pages

1. Buka repository di GitHub
2. Klik **Settings** (tab di atas)
3. Scroll ke bawah, cari menu **Pages** di sidebar kiri
4. Di bagian **Source**, pilih:
   - Branch: `main` (atau `master`)
   - Folder: `/ (root)`
5. Klik **Save**

## Langkah 3: Akses Website

Tunggu 1-2 menit, lalu akses website Anda di:

```
https://username.github.io/repository-name/
```

Ganti:
- `username` dengan username GitHub Anda
- `repository-name` dengan nama repository Anda

## Contoh

Jika username GitHub Anda adalah `johndoe` dan repository bernama `bplc-detector`, maka URL-nya:

```
https://johndoe.github.io/bplc-detector/
```
